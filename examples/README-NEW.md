## 🎯 Quick Start (Choose One)

### 🌐 For Web Development

**React (Fastest Setup)** - 5 minutes
```bash
cd node-react/backend && npm install && npm run dev
# Terminal 2:
cd node-react/frontend && npm install && npm run dev
# Visit http://localhost:5173
```
📚 Read: `node-react/README-DETAILED.md`

**Python + Vue** - 8 minutes
```bash
cd python-vue/backend && pip install -r requirements.txt && python app.py
# Terminal 2:
cd python-vue/frontend && npm install && npm run dev
```
📚 Read: `python-vue/README-DETAILED.md`

**Go + Svelte (Smallest Bundle)** - 5 minutes
```bash
cd go-svelte/backend && go run main.go
# Terminal 2:
cd go-svelte/frontend && npm install && npm run dev
```
📚 Read: `go-svelte/README-DETAILED.md`

**Java + Angular (Enterprise)** - 10 minutes
```bash
cd java-angular/backend && mvn spring-boot:run
# Terminal 2:
cd java-angular/frontend && npm start
```
📚 Read: `java-angular/README-DETAILED.md`

### 📱 For Mobile Development

**Flutter (iOS + Android + Web)** - 10 minutes
```bash
cd node-flutter/backend && npm install && npm run dev
# Terminal 2:
cd node-flutter/frontend && flutter pub get && flutter run
```
📚 Read: `node-flutter/README-DETAILED.md`

**Rust + Swift (iOS Native)** - 15 minutes
```bash
cd rust-swift/backend && cargo run
# Xcode:
cd rust-swift/frontend && open TodoApp.xcodeproj
```
📚 Read: `rust-swift/README-DETAILED.md`

**C# + Kotlin (Android Native)** - 12 minutes
```bash
cd csharp-kotlin/backend && dotnet run
# Android Studio:
cd csharp-kotlin/frontend && (Open in Android Studio)
```
📚 Read: `csharp-kotlin/README-DETAILED.md`

### 🔧 For Scripts & CLI Tools

**TypeScript (No Framework)** - 5 minutes
```bash
cd node-typescript/backend && npm install && npm run dev
# Terminal 2:
cd node-typescript/frontend && npm install && npx tsx src/index.ts
```
📚 Read: `node-typescript/README-DETAILED.md`

### 🏢 For Enterprise/Legacy

**PHP + Laravel** - 10 minutes
```bash
cd php-typescript/backend && composer install && php artisan serve
# Terminal 2:
cd php-typescript/frontend && npm install && npm start
```
📚 Read: `php-typescript/README-DETAILED.md`

## 💡 Pro Tips

- Each example is independent - copy the entire folder to start a new project
- Read the master guides first to understand Veld
- Use multiple terminals for backend and frontend
- Check browser DevTools Network tab to debug API calls
- Modify veld contracts and run `veld generate` to see types update
- All examples use the same API contract (Todo + User management)
- Type safety is enforced throughout - TypeScript catches errors early
