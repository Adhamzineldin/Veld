## 🔧 Common Commands

### Run Any Example
```bash
# Terminal 1 - Backend
cd <example>/backend
npm install && npm run dev      # JavaScript
python -m venv venv && source venv/bin/activate && pip install -r requirements.txt && python app.py  # Python
go run main.go                  # Go
mvn spring-boot:run            # Java
cargo run                       # Rust
php artisan serve              # PHP

# Terminal 2 - Frontend
cd <example>/frontend
npm install && npm run dev      # Web frameworks
flutter pub get && flutter run  # Flutter
# Open in Xcode/Android Studio  # iOS/Android native
```

### Generate Code
```bash
cd <example>/veld
veld generate
```
